
# 🎓 Экзаменационный Квест: Философия

Интерактивная игра для подготовки к экзамену по философии. 46 вопросов с развернутыми ответами, система самооценки и красивая статистика прогресса.

## 📋 Что нужно для запуска

- **Python 3.8 или выше** (проверить версию: `python --version`)
- **Git** (необязательно, если скачаешь архив вручную)

---

##  Пошаговая инструкция

### Шаг 1: Скачай проект

**Вариант А — через Git (рекомендуется):**
```bash
git clone https://github.com/KhAnnVit/exam_prep_game.git
cd exam-prep-game
```

**Вариант Б — вручную:**
1. Зайди на страницу репозитория на GitHub
2. Нажми зелёную кнопку **Code** → **Download ZIP**
3. Распакуй архив в удобную папку

---

### Шаг 2: Установи Python (если ещё не установлен)

1. Скачай Python с официального сайта: https://www.python.org/downloads/
2. При установке **обязательно** поставь галочку **"Add Python to PATH"**
3. После установки перезапусти компьютер

---

### Шаг 3: Переименуй файл с вопросами

️ **ВАЖНЫЙ ШАГ!** В папке проекта есть файл `exam_questions_start_copy.csv` — это шаблон с вопросами и ответами. Его нужно переименовать, чтобы игра его увидела.

**На Windows:**
1. Открой папку проекта
2. Найди файл `exam_questions_start_copy.csv`
3. Нажми на него правой кнопкой мыши → **Переименовать**
4. Измени имя на `exam_questions.csv` (убери `_start_copy`)
5. Подтверди изменение расширения, если Windows спросит

**Через командную строку:**
```bash
ren exam_questions_start_copy.csv exam_questions.csv
```

**На macOS/Linux:**
```bash
mv exam_questions_start_copy.csv exam_questions.csv
```

---

### Шаг 4: Создай виртуальное окружение и установи библиотеки

Открой терминал (Command Prompt или PowerShell) **в папке проекта** и выполни команды по очереди:

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

После активации виртуального окружения в начале строки терминала появится `(venv)`.

---

### Шаг 5: Запусти игру! 

```bash
python main.py
```

Откроется окно с игрой. Удачи на экзамене! 🍀

---

## 🎮 Как играть

1. **Вкладка "Игра"** — случайный вопрос из списка. Попробуй ответить вслух, затем нажми **"Показать ответ"** и оцени себя по шкале от 0 до 5.
2. Вопросы с оценкой **5** больше не появляются — ты их уже выучила!
3. **Вкладка "Статистика"** — показывает твой прогресс в виде heatmap и круговой диаграммы.

## ❗ Частые проблемы

**"Python не является внутренней или внешней командой"**
→ Python не добавлен в PATH. Переустанови Python с галочкой "Add to PATH".

**"No module named 'pandas'"**
→ Не установлены библиотеки. Выполни `pip install -r requirements.txt`

**"PermissionError: exam_questions.csv"**
→ Файл открыт в Excel. Закрой Excel и запусти игру снова.

**"Файл exam_questions.csv не найден"**
→ Ты не переименовала `exam_questions_start_copy.csv`. См. Шаг 3.

## 📊 Для продвинутых

Хочешь изменить вопросы или добавить свои? Открой `exam_questions.csv` в Excel и отредактируй колонки:
- **Number** — номер вопроса
- **Question** — текст вопроса
- **Answer** — развернутый ответ
- **Score** — оставь 0 (оценки сохраняются автоматически)
